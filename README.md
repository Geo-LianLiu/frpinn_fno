### A physics-informed deep learning framework for fast and accurate computation of layered medium Green’s functions (LMGF), combining Physics-Informed Neural Networks (PINNs) and Fourier Neural Operator (FNO).

#### Install
pip install torch numpy scipy

#### Applications
- Geophysical exploration (e.g., magnetotellurics)
- Electromagnetic wave propagation in layered media
- Green’s function surrogate modeling
- Fast forward modeling for inverse problems
